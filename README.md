# ☕ Coffee Shop Sales Dashboard

## 📌 Project Overview

This is my first Data Analytics project built using Microsoft Excel.

The dashboard analyzes coffee shop sales data and presents key business insights using interactive charts, Pivot Tables, and slicers.

## 🛠 Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* Data Cleaning
* Dashboard Design

## 📊 Dashboard Features

* 💰 Total Sales
* 👥 Total Footfall
* 🧾 Average Bill per Person
* 🛒 Average Orders per Person
* 📈 Hourly Sales Analysis
* 🏪 Store-wise Sales Performance
* ☕ Product Category Distribution
* 📅 Weekday Order Analysis
* ⭐ Top 5 Best Selling Products
